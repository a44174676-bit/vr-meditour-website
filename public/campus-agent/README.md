# AI안내에이젼트로봇시스템 MVP

## 개요
대학교 캠퍼스, 도서관, 기숙사에서 학생과 방문객을 돕는 AI 안내 에이젼트 웹·태블릿형 MVP입니다.

## 출원번호
10-2026-0080691

## 주요 기능
- 질문 의도 분류
- 캠퍼스 안내
- 도서관 안내
- 기숙사 안내
- 외국인 유학생 다국어 안내
- 장애 학생 접근성 안내
- 긴급호출 시뮬레이션
- 초기 화재 방재 시뮬레이션
- 환경정비·청소 시뮬레이션
- 관리자 대시보드 로그

## 실행 방법
index.html을 브라우저에서 열면 실행됩니다.
