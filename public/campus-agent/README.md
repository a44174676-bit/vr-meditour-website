# AI안내에이젼트로봇시스템 MVP

## 개요
대학교 캠퍼스, 도서관, 기숙사에서 학생과 방문객을 돕는 AI 안내 에이젼트 웹·태블릿형 MVP입니다.

## 출원번호
10-2026-0080691

## 주요 기능
- 질문 의도 분류
- 캠퍼스 안내
- 도서관 안내
- 기숙사 안내
- 외국인 유학생 다국어 안내
- 장애 학생 접근성 안내
- 긴급호출 시뮬레이션
- 초기 화재 방재 시뮬레이션
- 환경정비·청소 시뮬레이션
- 관리자 대시보드 로그

## 실행 방법
index.html을 브라우저에서 열면 실행됩니다.

## 파일 구조
- index.html
- styles.css
- script.js
- data.js
- README.md

## 향후 확장
- 실제 AI API 연동
- 학교 DB 연동
- 관리자 알림 API 연동
- 이동형 로봇 연동
- 센서 데이터 연동
- 청소 모듈 연동

## AR글라스 확장 개념
본 MVP는 실제 AR글라스를 직접 제어하지 않는 웹 기반 시연 버전입니다. 다만 부산외국어대학교 캠퍼스 건물 데이터와 층별 시설 정보를 기반으로, 향후 AR글라스에서 건물 마커, 이동 방향, 도착 후 층별 안내를 표시할 수 있도록 설계되었습니다.
